Monster by Kaitlyn Cortez

I was actually not excited about this project when it was first introduced to me because I don't like working with charcoal. I don't like how messy charcoal is and how it can get everywhere and I mean EVERYWHERE. I do love the end result of this project due to the dimensions that I created.

This image was one I made in my Drawing class. The size of this monster is on large drawing paper and is made using charcoal.
